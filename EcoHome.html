Aplikasi ini direka untuk membantu kanak-kanak mengenali jenis bahan buangan, belajar kitar semula melalui AR, model 3D dan kuiz.

Cara Guna Aplikasi

1. Tekan Mula untuk melihat AR.

2. Halakan kamera ke kad (image target) untuk melihat AR.Tekan model 3D untuk henti dan mainkan semula suara narasi serta aktifkan rotasi model 3D.Klik ikon Info untuk lihat pop-up maklumat.

3. Tekan butang menu Info untuk lihat fakta menarik.Tekan setiap kad untuk mendengar audio maklumat.

4. Jawab kuiz untuk uji pengetahuan anda.
